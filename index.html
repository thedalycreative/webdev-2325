  • Deployment preview using Live Server "Go Live"

  • Built full page structure with header, main, footer, sections
  • Reinforced accessible navigation and `alt` text best practices
  • Students recreated demo layout + added inline comments explaining intent

Lesson 4 • CSS & HTML Deep Dive
  • CSS Fundamentals: Three types (inline, internal, external)
  • Selectors: Element, class (.navbar), styling properties
  • Box Model: Content, padding, border, margin
  • HTML Structure: Nav, tables, semantic elements
  • Advanced CSS: Hover effects, button styling, color systems
Resources: HTML Tags • CSS Docs
Lesson 5 • Client Relations & Performance
  • Client Relations: Real-world client assessment focus
  • Case Study: Swan Valley Cellars e-commerce optimization
  • Performance Issues: Core Web Vitals, LCP, mobile abandonment
  • Solutions: Caching, image compression, hosting migration
  • Platform Research: Wix, Shopify, Kinsta, WP Engine comparison
Key concept: Website performance directly impacts revenue & SEO rankings
Fluff'n'Stuff Case Study
Lesson 6 • QA & Accessibility
  • CSS Exercise: Text decorations and styling properties
  • QA Best Practices: Checklists, risk assessments, functional testing
  • WCAG Compliance: Keyboard navigation, alt text, contrast ratios
  • Testing Tools: Accessibility checkers, browser extensions, AI tools
  • Client Communication: Simplifying technical issues, solution-focused reporting
Hands-on: Used ChatGPT/Gemini for QA testing, accessibility checker showed 53% compliance
View QA Checklist
Lesson 7 • Tech Support & SEO
  • CSS Practice: Table styling, borders, padding/margin properties
  • Tech Support Role-play: Client communication and troubleshooting
  • SEO Optimization: Metadata, keywords, page titles, search visibility
  • Navigation Fixes: HTML href attributes, broken links, 404 pages
  • Service Management: SLA timelines, NPS feedback, documentation
Case studies: Katie (SEO issues), Marvin (navigation links), implementing QA checklists
Katie & Marvin Case Studies
Lesson 8 • Flexbox & Google Fonts
  • Google Fonts: Importing and applying custom fonts from fonts.google.com
  • CSS Variables: Using :root selector for color schemes and reusable values
  • Flexbox Fundamentals: Flex containers, main/cross axis, flex-direction properties
  • Flexbox Properties: justify-content, space-between, space-around, alignment
  • Bootstrap Intro: Pre-built components, responsive design, accessibility features
Interactive: Flexbox Froggy • CSS Diner • Navigation bar styling • Flexbox vs Grid layouts
Lesson 9 • Bootstrap Framework Introduction
  • Bootstrap Overview: Pre-defined styles, responsive design, and accessibility features
  • Bootstrap Templates: Using ready-made templates and classes (W3Schools, Bootstrap docs)
  • Responsive Design: Screen size categories (small, medium, large, extra large)
  • Bootstrap Components: Navigation bars, buttons, containers, and pre-built elements
  • Project Start: Sweetify dessert cafe website with centered logo navbar
